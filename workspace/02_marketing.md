## 3. CTA戦略

| 項目 | 内容 |
|---|---|
| テキスト | **「Instagramをフォローする」** |
| 配置 | キャッチコピーの直下。画面中央。ファーストビュー内に必ず収める。 |
| デザイン指示 | 背景色: ウォームブラウン `#6B4C35`／テキスト: オフホワイト `#FAF7F2`／角丸: 4px／パディング: 16px 40px／フォントサイズ: 16px Bold |

| サブCTA配置 | メインCTAの右横（PC）/ 直下（スマホ）。アウトラインボタンで視覚的優先度を下げる。 |
| サブCTAデザイン指示 | ボーダー色: `#6B4C35`／テキスト: `#6B4C35`／背景: 透明 |

**設計意図:** Instagramフォロワー獲得を最優先KPIとする。来店CTAは存在感を持たせながらも視覚的に二番手に置き、ユーザーの迷いを防ぐ。

## 5. ソーシャルプルーフ設計

### 証言パターン設計

お客様の声セクションでは、**3タイプの証言**を組み合わせることで多様なターゲット層に刺さる設計とする。

#### タイプA — 習慣化証言（リピーター訴求）

```
「週2回通っています。コーヒーの香りで一週間がリセットされる気がして、
もはや生活の一部になっています。」
— Ayaka（28歳・会社員）
```

設計ポイント:
- 頻度の具体性（週2回）で信頼度を上げる
- 「生活の一部」でブランドへの統合感を示す

#### タイプB — SNS拡散証言（Instagram誘導と連動）

```
「バスクチーズケーキの写真をInstagramに投稿したら、友達から"どこ?"って
メッセージが50件来ました。それ以来、来るたびに投稿しています（笑）」
— Mana（25歳・フリーランス）
```

設計ポイント:
- 投稿行動の自然な描写でUGC（ユーザー生成コンテンツ）文化を印象づける
- 「友達への連絡」で口コミ拡散のリアルを見せる

#### タイプC — 人的温かみ証言（スタッフ・居心地訴求）

```
「顔を覚えてもらえて、毎回"おかえりなさい"って言ってもらえるのが
嬉しくて。このお店に来ると、家に帰ってきた感じがするんです。」
— Hiromi（32歳・主婦）
```

設計ポイント:
- 「おかえりなさい」というキーワードで感情的な帰属感を喚起
- 地域密着・常連文化のリアルな側面を証言者の言葉で伝える

## 6. 信頼シグナル

### 「地域密着5年」の表示設計

| タイミング | 方法 | 文言例 |
|---|---|---|
| コンセプトセクション | テキストバッジ or インラインテキスト | 「2021年のオープンから、この街で5年。」 |
| お客様の声セクション | 数字コンポーネント（CounterUI） | **「5年」** — 地域に根ざした歴史 |
| フッター | 著作権表記のそば | 「THE CORNER CAFE — Since 2021」 |

### 「リピーター率80%以上」の表示設計

| タイミング | 方法 | 文言例 |
|---|---|---|
| お客様の声セクション冒頭 | 大きな数字＋補足テキスト（CounterUI） | **「80%以上」** のお客様が、また来てくれる |
| ヒーローセクション サブテキスト | さりげなく混ぜ込む | 「地域密着5年、リピーターに愛されるカフェ」 |
| Googleレビュー連携 | 星評価コンポーネント | 高評価スコアと口コミ件数を並列表示 |

### 信頼シグナル表示のタイミング設計（スクロール順）

```
ヒーロー        → USPとしてさらっと触れる（主張ではなく前提として）
  ↓
コンセプト      → 「5年」という歴史をストーリーとして語る
  ↓
お客様の声      → 「80%以上」「5年」を数字で視覚化する（最も効果的なタイミング）
  ↓
フッター        → 「Since 2021」で静かに念押しする
```

**設計原則:** 信頼シグナルは「自慢」ではなく「事実の提示」として機能させる。数字は大きく見せるが、煽らない。トーンはあくまで温かく・謙虚に。

## 付録: コピーライティング禁止事項・推奨事項

### 禁止表現（ブランドボイス違反）

- 「今だけ！」「期間限定！」などの緊急性煽り
- 「業界最高」「No.1」などの根拠のない最上級表現
- 割引強調（「〇〇円OFF！」「半額！」）
- 過度な感嘆符の連続使用（「！！！」）
- カタカナ多用の外来語羅列（「コスパ最強」「クオリティ」「マスト」など多用は避ける）

### 推奨表現（ブランドボイス適合）

- 「ほっとする」「一息つく」「ほんとうにおいしい」
- 「こだわり」「丁寧に」「毎日来たくなる」
- 「あなたの」「毎日の」「日常の」（個人への語りかけ）
- 「手作り」「自家焙煎」「豆から選ぶ」（プロセスの開示）
- 「また来てください」「お待ちしています」（温かい待ち受け姿勢）
