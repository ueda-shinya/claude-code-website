# THE CORNER CAFE — 仕様書

## 1. ビジネス目標

| 項目 | 内容 |
|------|------|
| 主目的 | 認知拡大＋ファン化。こだわりのコーヒーと世界観を発信してSNSシェアを促進 |
| メインCTA | Instagramフォロー／来店 |
| サブCTA | お問い合わせ・予約 |
| KPI | SNSフォロワー増加、月間来店者数アップ |

## 2. ターゲット

| 項目 | 内容 |
|------|------|
| メインペルソナ | 20〜30代女性・カフェ好き。おしゃれなカフェをSNSで発信、スイーツやラテアートに敏感 |
| 職業 | OL・フリーランス・学生 |
| 悩み | 「本当においしいコーヒーが飲めるカフェ」「映える・落ち着ける空間」を探している |
| 求めるもの | 非日常の癒し、SNSに投稿したくなる体験、本物のコーヒーの味 |

## 3. バリュープロポジション

**「毎日来たくなる、本物の一杯と空間」**

- 自家焙煎コーヒー × 手作りスイーツの本格クオリティ
- 地域密着5年、リピーター率80%以上の実績
- 素材・味・空間すべてにこだわる上質な体験

| # | セクション | 目的 |

| 項目 | 内容 |

| メインカラー | ウォームブラウン `#6B4C35` |

## 6. 機能要件

- お問い合わせフォーム（PHP送信、セキュリティ全実装）
- スムーズスクロール
- スクロールリビールアニメーション
- ハンバーガーメニュー（スマホ）
- FAQアコーディオン
- メニュータブ切り替え（コーヒー／スイーツ）
- フォームバリデーション（フロント＋バック）
- レート制限・Honeypot

## 7. SEO要件

- ターゲットキーワード: カフェ / 自家焙煎コーヒー / 手作りスイーツ
- OGP / Twitter Card 設定（Instagram向け）
- JSON-LD構造化データ（CafeOrRestaurant / FAQPage / WebSite）
- Googleビジネスプロフィール連携を意識したローカルSEO

## 8. ブランドボイス・トーン

- **温かく、等身大に。** 高級感を押し付けず、「ご近所さんに語りかける」トーン
- 専門用語は使うが、難しくしない（「スペシャルティコーヒー」は使うが解説を添える）
- SNSシェアを意識した短くキャッチーな見出し
- 禁止: 過度な敬語、セールス臭の強い言葉、過剰な！感嘆符

## 9. 技術仕様

- 静的HTML（index.html）+ CSS（styles.css）+ JS（script.js）+ PHP（form_handler.php）
- モバイルファースト、ブレークポイント: 375px / 768px / 1280px
- 外部ライブラリ禁止（バニラのみ）
- FLOCSS命名規則
