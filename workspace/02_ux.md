### ブレークポイント別レイアウト変化詳細

#### 375px（SP）基本設計
- コンテナ: `width: 100%`, `padding: 0 24px`
- ナビ: ハンバーガーアイコン表示、全画面ドロワー（`position: fixed`, `top: 0`, `left: 0`, `width: 100%`, `height: 100dvh`）
- ヒーロー: 1カラム縦積み、テキスト中央揃え、`min-height: 100dvh`
- メニューカード: 1カラム（`grid-template-columns: 1fr`）
- ギャラリー: 2カラム（`grid-template-columns: repeat(2, 1fr)`）
- スタッフ: 1カラム縦積み
- FAQ: アコーディオン縦1列
- 店舗情報: 1カラム縦積み（地図→テキスト順）

#### 768px（TAB）変化点
- コンテナ: `max-width: 768px`, `padding: 0 48px`
- ナビ: ハンバーガー継続（タブレットもドロワー式維持）
- ヒーロー: テキスト左揃えに変更、`min-height: 85vh`
- メニューカード: 2カラム（`grid-template-columns: repeat(2, 1fr)`）
- ギャラリー: 3カラム（`grid-template-columns: repeat(3, 1fr)`）
- スタッフ: 2カラム（`grid-template-columns: repeat(2, 1fr)`）
- FAQ: 2カラム分割（左:質問一覧、右:回答エリア） ※ アコーディオン継続
- 店舗情報: 2カラム（テキスト:地図 = 1fr:1fr）
- コンセプト: 画像＋テキスト横並び（`grid-template-columns: 1fr 1fr`）

#### 1280px（PC）変化点
- コンテナ: `max-width: 1080px`, `margin: 0 auto`, `padding: 0 64px`
- ナビ: ハンバーガー非表示、水平インラインメニュー表示
- ヒーロー: 左右分割レイアウト（テキスト40% : 画像60%）、`min-height: 100vh`
- メニューカード: 3カラム（`grid-template-columns: repeat(3, 1fr)`）
- ギャラリー: 4カラム（`grid-template-columns: repeat(4, 1fr)`） + 1枚フィーチャー
- スタッフ: 3カラム（`grid-template-columns: repeat(3, 1fr)`）
- FAQ: 2カラム（`grid-template-columns: 2fr 3fr`）
- 店舗情報: 2カラム（テキスト:地図 = 5fr:7fr）
- コンセプト: 画像＋テキスト横並び（`grid-template-columns: 5fr 7fr`）

## 8. アイコン・デコレーション

### 8-1. SVGアイコン方針

- **使用ライブラリ:** インラインSVGのみ（外部ライブラリ不使用。ページ軽量化・色変更の容易性）
- **サイズ:** 基本 `24×24px`、ナビ等小型 `20×20px`、大型装飾 `48×48px`
- **色指定:** `currentColor` を使用し、CSS の `color` プロパティで制御
- **アクセシビリティ:** 意味のあるアイコンには `aria-label`、装飾のみは `aria-hidden="true"`

#### 使用アイコン一覧

| 用途               | アイコン形状           | サイズ      |
|------------------|--------------------|-----------|
| ハンバーガーメニュー  | 三本線 / ×           | 24×24px   |
| Instagram リンク   | Instagram ロゴ枠線    | 24×24px   |
| MAP ピン          | Location pin        | 20×20px   |
| 電話              | Phone outline       | 20×20px   |
| 時計（営業時間）    | Clock outline       | 20×20px   |
| 定休日（カレンダー） | Calendar outline    | 20×20px   |
| FAQ ＋/×          | Plus / cross        | 20×20px   |

| 星（評価）          | Star filled         | 16×16px   |

## 付録：実装チェックリスト

| 項目                              | 備考                                        |
|----------------------------------|---------------------------------------------|
| CSS変数を `:root` に一括定義       | `styles.css` 冒頭に配置                      |
| Google Fonts `@import` をCSSトップ | Preconnect を HTML `<head>` に追加           |
| IntersectionObserver しきい値      | `threshold: 0.15`, `rootMargin: '0px 0px -40px 0px'` |
| フォームバリデーション              | HTML5 required + JS追加チェック              |
| `prefers-reduced-motion` 対応     | アニメーション全体に適用                      |
| `focus-visible` スタイル           | キーボード操作UXのためアウトライン維持          |
| 画像 `loading="lazy"`             | ヒーロー画像のみ `eager`                     |
| `alt` テキスト                    | 全画像に日本語 alt 記述                       |
